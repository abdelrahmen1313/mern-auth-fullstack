# AUTHONE PROJECT

This project demonstrates a complete user authentication operations including

**TASKS DONE**

1. SignIn
Loggin in into the application with valid credentials
2. SignUp
Creating a new account and gaining limited access to the application
3. Email Verification
Email verification operation linked to an otp service with tight cors allowance and client fingerprinting to mitigate/reduce bot accounts
4. Reset Password
Allowing users to reset their passwords via signed email links


**PENDING TASKS**

1. Trusted devices (sudo mode)
allowing a signed in user to sign a trusted session client, basically a browser for sudo operations that bypass 2fa
2. Session Monitoring
Allowing users to visualize their active sessions, and remotely logout on trusted devices
